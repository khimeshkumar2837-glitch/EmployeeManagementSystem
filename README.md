# Employee Management System

## Project Overview
This is a Java-based Employee Management System using ArrayList and HashMap.
The project supports CRUD operations, file handling, searching, and reporting.

## Features
- Add Employee
- View Employees
- Search Employee
- Save Data to File
- Exception Handling

## Technologies Used
- Java
- ArrayList
- HashMap
- File Handling
- Serialization

## How to Run
1. Open project in VS Code or IntelliJ
2. Compile:
   javac EmployeeManagementSystem.java
3. Run:
   java EmployeeManagementSystem
